### [Honey Badger AI](https://github.com/ShivamBhosale/Honey-Badger---AI-Assistant) — Reliability-First GenAI System
Multi-stage AI validation pipeline where every response is generated, critiqued, revised, rule-evaluated, confidence-scored, and guardrailed before being shown. Fully local via Ollama — no API key required.

`GenAI` `Streamlit` `Ollama` `LLM` `Python`

### [Ask-Your-PDF](https://github.com/ShivamBhosale/Ask-you-pdf) — RAG Document Q&A
Retrieval-Augmented Generation app that lets users query any PDF in natural language. Chunks documents, builds FAISS vector embeddings, and forces the LLM to answer strictly from retrieved context — no hallucinations.

`Python` `RAG` `FAISS` `LangChain` `LLM` `Vector Search`
